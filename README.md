# Brave-appimage

Unofficial AppImages for [Brave Web Browser](https://brave.com), built automatically for **x86_64** and **aarch64**.

Syncs with upstream Brave releases every 4 hours via GitHub Actions.

## Download

Go to [**Releases**](https://github.com/thisisarnabdas/Brave-appimage/releases/tag/latest) and grab the AppImage for your architecture:

| Architecture | File |
|---|---|
| x86_64 (most desktops/laptops) | `Brave-Web-Browser-vX.X.X-x86_64.AppImage` |
| aarch64 (ARM, e.g. Raspberry Pi 4) | `Brave-Web-Browser-vX.X.X-aarch64.AppImage` |

## Usage

```bash
chmod a+x Brave-Web-Browser-*-x86_64.AppImage
./Brave-Web-Browser-*-x86_64.AppImage
```
